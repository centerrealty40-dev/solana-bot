### §3.2 Какие события участвуют

Спека допускает путь **(A)** зеркальные `live_position_*` или **(B)** вывод только из `execution_attempt`/`execution_result`.

| Требование | Статус | Заметки |
|------------|--------|---------|
| Путь **(A)** — `live_position_*` | ✅ | Запись в [`tracker.ts`](../../../src/papertrader/executor/tracker.ts), [`main.ts`](../../../src/papertrader/main.ts) (open); Zod в [`events.ts`](../../../src/live/events.ts) |
| Путь **(B)** | ☐ | Не реализован (и не требуется, если зафиксирован выбор **A** в CHANGELOG) |
| Участие `execution_attempt` / `execution_result` в replay | ☐ | Не используется для восстановления `open` |

### §3.3 Pending intents (attempt без result)

| Требование | Статус | Заметки |
|------------|--------|---------|
| Незавершённый intent не должен тихо ломать `open` | 🟡 | Для модели **live_position_*** pending swap не отражается, пока не записан следующий снимок позиции; отдельное событие `reconcile_pending` / timeout — **нет** |
