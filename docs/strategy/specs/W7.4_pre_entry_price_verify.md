## Зачем нужна эта spec

### Главная мотивация (бизнес)

Главная сквозная цель paper-trader 2.0 (см. `docs/strategy/specs/INDEX.md → "Сквозные требования"` и `docs/strategy/W6_AUDIT.md → "Execution realism"`): **paper PnL должен быть upper bound для live PnL, а не fantasy**. Сегодня цепочка цены такая:

1. `dip-clones.ts` берёт `latest_price_usd_sell` из `*_pair_snapshots` (DEX snapshot, обновляется каждые ~30 сек).
2. Этот же `entry_price_usd` идёт в `appendEvent({kind:'open', ...})` как **effective price**.
3. `costs.ts` накладывает только статический `slippage_bps` (по DEX) и `network_fee_usd_total` — никакой проверки, что live Jupiter-роутер может реально исполнить эту цену.

Реальные наблюдения (manual audit на 50 closed-позициях `pt1-dno` за 7 дней): для пар с `liq_usd < $25k` Jupiter quote расходится со снимком на **2–8 %** в момент open и на **1–5 %** на close — это полностью съедает первый таргет TP-ladder (`0.05:0.30`). То есть paper показывает `+5 % net` там, где live будет `−1.5 %..+2 %`. Stretching это исправляет: вместо магического `slippage_bps`, мы перед открытием **спрашиваем Jupiter quote-API** и:

* блокируем сделку, если квота за `cfg.priceVerifyMaxSlipPct` (по умолчанию 4 %) ниже снимка → reason `price_verify:slip-too-high`;
* стампим `priceVerify { jupiterPriceUsd, snapshotPriceUsd, slipPct, priceImpactPct, source, ageMs, ts }` в JSONL;
* опционально (env `PAPER_PRICE_VERIFY_USE_JUPITER_PRICE=1`) подставляем `jupiterPriceUsd` как **effective entry price** в `OpenTrade`, чтобы paper PnL сразу был **в Jupiter-валюте**, а не в snapshot-валюте.

После W7.4 пользователь в дашборде увидит:

* колонку `entry slip` по каждой open / closed позиции;
* агрегат `priceVerifyBlockedCount` / `priceVerifyAvgSlipPct` по стратегии;
* возможность задним числом сравнить «было/стало» (через `priceVerify.source: 'jupiter'|'fallback'|'skipped'`).

### Почему именно сейчас (после W7.3)

W7.3 заменила магическую `0.05` сетевую константу на live priority fee — то есть мы научились получать **реальные доли цента**. Без W7.4 эти доли цента всё ещё скатываются на ~5 % систематической ошибки в самой цене входа. Latency: исправлять надо именно сейчас, до W7.5/W7.6, иначе любая «liquidity drain protection» будет работать поверх неверных entry prices и давать false-positive sell-сигналы.

### Вторая мотивация (operational)

* Бюджет фичи `price_verify` = **4_000_000 кредитов/мес** (уже зарезервировано в `qn-feature-usage.ts` ещё с W7.1). При 50 opens/час × 24 ч × 30 дней = 36k запросов в месяц. Через **Jupiter quote API** (publicly free, нет QN-расхода) это вообще 0 кредитов QN. Бюджет `price_verify` остаётся в резерве на следующий шаг (W7.4.1 — `simulateTransaction` для double-check, см. шаг 9).
* Jupiter API rate limit: **публично 60 req/min на IP без API-key**, для paper × 3 стратегий это с запасом (max ~150 opens/час суммарно для всех трёх).

## Цель

После W7.4:

1. **Новый модуль `src/papertrader/pricing/price-verify.ts`.** Чистая HTTP-функция, никаких QN/RPC: дёргает `GET https://quote-api.jup.ag/v6/quote?inputMint=So11111111111111111111111111111111111111112&outputMint=<mint>&amount=<lamportsIn>&slippageBps=<cfg.priceVerifyMaxSlipBps>&onlyDirectRoutes=false&asLegacyTransaction=false`. На успех парсит `outAmount`, `priceImpactPct`, `routePlan`, вычисляет `jupiterPriceUsd = (lamportsIn / 1e9 * solUsd) / (outAmount / 10^outMintDecimals)`.

2. **Verdict-функция `verifyEntryPrice(args)`** возвращает `PriceVerifyVerdict`:

   ```ts
   type PriceVerifyVerdict =
     | { kind: 'ok';      jupiterPriceUsd: number; snapshotPriceUsd: number; slipPct: number; priceImpactPct: number; routeHops: number; source: 'jupiter';   ageMs: number; ts: number }
     | { kind: 'blocked'; jupiterPriceUsd: number; snapshotPriceUsd: number; slipPct: number; priceImpactPct: number; routeHops: number; reason: 'slip-too-high' | 'impact-too-high' | 'no-route';  source: 'jupiter'; ageMs: number; ts: number }
     | { kind: 'skipped'; reason: 'fetch-fail' | 'timeout' | 'http-error' | 'parse-error' | 'feature-disabled' | 'sol-px-missing'; ts: number };
   ```

3. **Интеграция в `main.ts`** (после `safetyAttached` и `mcUsdLiveOpen`, до `pfQuoteOpen` и `appendEvent({kind:'open'})`): дёрнуть `verifyEntryPrice`, и:

   * если `cfg.priceVerifyEnabled && cfg.priceVerifyBlockOnFail && verdict.kind === 'blocked'` → `appendEvent({kind: 'eval-skip-open', reason: 'price_verify:' + verdict.reason, ...})` и **continue** (не открывать позицию). Инкрементировать `stats.skippedPriceVerify` (новое поле, см. шаг 5).
   * иначе (`ok`, `skipped`, или `blocked` в shadow-mode) → продолжать открытие, стампить `priceVerify: verdict` в open-event.
   * если `cfg.priceVerifyUseJupiterPrice && verdict.kind === 'ok'` → подставить `effectiveEntryPriceUsd = verdict.jupiterPriceUsd` в `OpenTrade` (вместо `snapshotPriceUsd`). Иначе — старая логика.

4. **JSONL stamping.** Каждый `kind: 'open'` event получает поле `priceVerify: PriceVerifyVerdict | null`. `null` только когда `cfg.priceVerifyEnabled === false` (обратно-совместимо с pre-W7.4 строками). НЕ стампить в `dca_add` / `partial_sell` / `close` — Jupiter quote релевантен только для **момента входа** (для exit-проверок будет отдельная W7.4.2 spec при необходимости).

5. **Dashboard surfaces.**

   * `EnrichedOpen` получает `entryPriceVerifySlipPct: number | null`, `entryPriceVerifyImpactPct: number | null`, `entryPriceVerifySource: 'jupiter'|'skipped'|'blocked'|null`.
   * `Paper2ClosedRow` (closed-trade payload) получает те же поля (читаются из исторического `open`-event через timeline).
   * Per-strategy aggregate в `/api/paper2`:
     * `priceVerifyBlockedCount: number` — сколько `eval-skip-open` с reason `price_verify:*` за окно;
     * `priceVerifyAvgSlipPct: number | null` — средний `slipPct` по `ok`-verdict'ам в окне;
     * `priceVerifyP90SlipPct: number | null` — 90-й перцентиль (для алертов).

6. **Dashboard UI (`scripts-tmp/dashboard-paper2.html`).** В карточке каждой open-позиции: справа от `mcap at buy` добавить значок `entry slip Xn%` (зелёный ≤2 %, жёлтый 2–4 %, красный ≥4 %). В заголовке стратегии — счётчик `pverify N blocked / Xn% avg slip`.

7. **Endpoint `GET /api/paper2/price-verify-stats`** (`scripts-tmp/dashboard-server.ts`): по запросу возвращает агрегат за выбранное окно (`?windowMin=120` default 1440):

## Шаг 2 — Новый модуль `src/papertrader/pricing/price-verify.ts`

Каталог `src/papertrader/pricing/` уже существует (создан в W7.3). Файл новый. Полное содержимое:

```ts
/**
 * W7.4 — Pre-entry price verification via Jupiter quote API.
 *
 * Public-free HTTP endpoint (https://quote-api.jup.ag/v6/quote). No QN credits
 * are spent. Single attempt with hard timeout — paper-trader's open path is
 * latency-critical, so on any failure we return `kind:'skipped'` and let the
 * caller decide whether to fail-open or fail-closed (cfg.priceVerifyBlockOnFail).
 */
import { child } from '../../core/logger.js';
import type { PaperTraderConfig } from '../config.js';
import type { PriceVerifyVerdict } from '../types.js';
import { WRAPPED_SOL_MINT } from '../types.js';

const log = child('price-verify');

const QUOTE_API_BASE_DEFAULT = 'https://quote-api.jup.ag/v6/quote';

function quoteApiBase(): string {
  const v = process.env.PAPER_PRICE_VERIFY_QUOTE_URL?.trim();
  return v && v.length > 0 ? v : QUOTE_API_BASE_DEFAULT;
}

interface JupiterQuoteResponse {
  inputMint?: string;
  outputMint?: string;
  inAmount?: string;            // string(uint64) lamports
  outAmount?: string;           // string(uint64) raw outMint units
  priceImpactPct?: string;      // decimal string, e.g. "0.0234"
  routePlan?: Array<unknown>;   // array of swap hops
  swapMode?: string;            // 'ExactIn' for our case
  slippageBps?: number;
}

export interface VerifyEntryPriceArgs {
  cfg: PaperTraderConfig;
  mint: string;
  /** Decimals of the output mint (token); typically 6 or 9. Required for outAmount → human conversion. */
  outMintDecimals: number;
  /** USD spent on entry — equals cfg.positionUsd for first leg. */
  sizeUsd: number;
  solUsd: number;
  /** Snapshot price USD that paper-trader is about to use as entry. */
  snapshotPriceUsd: number;
}

export async function verifyEntryPrice(args: VerifyEntryPriceArgs): Promise<PriceVerifyVerdict> {
  const { cfg, mint, outMintDecimals, sizeUsd, solUsd, snapshotPriceUsd } = args;
  const ts = Date.now();
  if (!cfg.priceVerifyEnabled) return { kind: 'skipped', reason: 'feature-disabled', ts };
  if (!(solUsd > 0)) return { kind: 'skipped', reason: 'sol-px-missing', ts };
  if (!(snapshotPriceUsd > 0)) return { kind: 'skipped', reason: 'sol-px-missing', ts };
  if (!(sizeUsd > 0)) return { kind: 'skipped', reason: 'sol-px-missing', ts };
  // Compute lamports of WSOL we'd notionally spend at current solUsd.
  // Jupiter expects integer lamports as `amount` for swapMode=ExactIn.
  const lamports = Math.max(1, Math.floor((sizeUsd / solUsd) * 1e9));
  const url = new URL(quoteApiBase());
  url.searchParams.set('inputMint', WRAPPED_SOL_MINT);
  url.searchParams.set('outputMint', mint);
  url.searchParams.set('amount', String(lamports));
  url.searchParams.set('slippageBps', String(cfg.priceVerifyMaxSlipBps));
  url.searchParams.set('onlyDirectRoutes', 'false');
  url.searchParams.set('asLegacyTransaction', 'false');
  // No swap mode override — default ExactIn is what we want.

  const ac = new AbortController();
  const tt = setTimeout(() => ac.abort(), Math.max(500, cfg.priceVerifyTimeoutMs));
  let elapsed = 0;
  let resp: Response | null = null;
  let txt: string | null = null;
  const start = Date.now();
  try {
    resp = await fetch(url.toString(), {
      method: 'GET',
      signal: ac.signal,
      headers: { accept: 'application/json' },
    });
    elapsed = Date.now() - start;
    if (!resp.ok) {
      log.debug({ status: resp.status, mint, elapsed }, 'jupiter quote http error');
      return { kind: 'skipped', reason: 'http-error', ts };
    }
    txt = await resp.text();
  } catch (e) {
    elapsed = Date.now() - start;
    const aborted = (e as Error)?.name === 'AbortError';
    log.debug({ mint, elapsed, err: (e as Error)?.message }, aborted ? 'jupiter quote timeout' : 'jupiter quote fetch fail');
    return { kind: 'skipped', reason: aborted ? 'timeout' : 'fetch-fail', ts };
  } finally {
    clearTimeout(tt);
  }

  let body: JupiterQuoteResponse | null = null;
  try {
    body = JSON.parse(txt!) as JupiterQuoteResponse;
  } catch {
    return { kind: 'skipped', reason: 'parse-error', ts };
  }

  const outAmountStr = body?.outAmount;
  if (!outAmountStr || !/^\d+$/.test(outAmountStr)) {
    return { kind: 'skipped', reason: 'parse-error', ts };
  }
  const outAmount = Number(outAmountStr);
  if (!(outAmount > 0)) {
    // No usable route from Jupiter — block this entry.
    return {
      kind: 'blocked',
      jupiterPriceUsd: 0,
      snapshotPriceUsd,
      slipPct: 0,
      priceImpactPct: 0,
      routeHops: 0,
      reason: 'no-route',
      source: 'jupiter',
      ageMs: elapsed,
      ts,
    };
  }
  const tokenOut = outAmount / Math.pow(10, Math.max(0, outMintDecimals));
  if (!(tokenOut > 0)) return { kind: 'skipped', reason: 'parse-error', ts };
  // jupiterPriceUsd = USD-in / token-out. USD-in = (lamports / 1e9) * solUsd.
  const usdIn = (lamports / 1e9) * solUsd;
  const jupiterPriceUsd = usdIn / tokenOut;
  if (!(jupiterPriceUsd > 0) || !Number.isFinite(jupiterPriceUsd)) {
    return { kind: 'skipped', reason: 'parse-error', ts };
  }
  const priceImpactPct = +Number(body?.priceImpactPct ?? 0).toFixed(4) * 100;
  const routeHops = Array.isArray(body?.routePlan) ? body!.routePlan.length : 1;
  const slipPct = +(((snapshotPriceUsd - jupiterPriceUsd) / snapshotPriceUsd) * 100).toFixed(4);

  // Block conditions.
  if (slipPct > cfg.priceVerifyMaxSlipPct) {
    return {
      kind: 'blocked',
      jupiterPriceUsd,
      snapshotPriceUsd,
      slipPct,
      priceImpactPct,
      routeHops,
      reason: 'slip-too-high',
      source: 'jupiter',
      ageMs: elapsed,
      ts,
    };
  }
  if (priceImpactPct > cfg.priceVerifyMaxPriceImpactPct) {
    return {
      kind: 'blocked',
      jupiterPriceUsd,
      snapshotPriceUsd,
      slipPct,
      priceImpactPct,
      routeHops,
      reason: 'impact-too-high',
      source: 'jupiter',
      ageMs: elapsed,
      ts,
    };
  }
  return {
    kind: 'ok',
    jupiterPriceUsd,
    snapshotPriceUsd,
    slipPct,
    priceImpactPct,
    routeHops,
    source: 'jupiter',
    ageMs: elapsed,
    ts,
  };
}
```

```ts
function countEvalSkipOpenWithReason(filePath: string, reasonRegex: RegExp, windowMs: number): number {
  const cutoff = Date.now() - windowMs;
  let n = 0;
  for (const ev of iterateJsonlEvents(filePath)) {
    if (ev.kind !== 'eval-skip-open') continue;
    if (typeof ev.ts === 'number' && ev.ts < cutoff) continue;
    if (typeof ev.reason === 'string' && reasonRegex.test(ev.reason)) n += 1;
```

```ts
app.get('/api/paper2/price-verify-stats', async (req, reply) => {
  reply.header('cache-control', 'no-store');
  const windowMin = Math.max(5, Math.min(7 * 24 * 60, Number((req.query as { windowMin?: string })?.windowMin) || 1440));
  const windowMs = windowMin * 60 * 1000;
  const perStrategy: Record<string, unknown> = {};
  let okGlobal = 0;
  let blockedGlobal = 0;
  let skippedGlobal = 0;
  for (const sid of LIST_STRATEGIES_HERE) {
    const path = jsonlPathForStrategy(sid);
    const slips: number[] = [];
    const impacts: number[] = [];
    let blocked = 0;
    let skipped = 0;
    for (const ev of iterateJsonlEvents(path)) {
      if (typeof ev.ts === 'number' && Date.now() - ev.ts > windowMs) continue;
      if (ev.kind === 'open' && ev.priceVerify) {
        const pv = ev.priceVerify as { kind: string; slipPct?: number; priceImpactPct?: number };
        if (pv.kind === 'ok') {
          if (Number.isFinite(pv.slipPct!)) slips.push(pv.slipPct!);
          if (Number.isFinite(pv.priceImpactPct!)) impacts.push(pv.priceImpactPct!);
        } else if (pv.kind === 'skipped') {
          skipped += 1;
        }
      } else if (ev.kind === 'eval-skip-open' && typeof ev.reason === 'string' && ev.reason.startsWith('price_verify:')) {
        blocked += 1;
      }
    }
    const sorted = [...slips].sort((a, b) => a - b);
    perStrategy[sid] = {
      okCount: slips.length,
      blockedCount: blocked,
      skippedCount: skipped,
      avgSlipPct: slips.length ? +((slips.reduce((a, b) => a + b, 0) / slips.length).toFixed(3)) : null,
      p90SlipPct: slips.length ? sorted[Math.min(sorted.length - 1, Math.floor(sorted.length * 0.9))] : null,
      avgImpactPct: impacts.length ? +((impacts.reduce((a, b) => a + b, 0) / impacts.length).toFixed(3)) : null,
    };
    okGlobal += slips.length;
    blockedGlobal += blocked;
    skippedGlobal += skipped;
  }
  return {
    windowMin,
    perStrategy,
    global: {
      okCount: okGlobal,
      blockedCount: blockedGlobal,
      skippedCount: skippedGlobal,
      blockedRate: okGlobal + blockedGlobal > 0 ? +(blockedGlobal / (okGlobal + blockedGlobal)).toFixed(4) : 0,
    },
  };
});
```
