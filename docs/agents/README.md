# Агент-доки в репозитории solana-bot

Каноническая копия материалов для AI/интегратора живёт **здесь**, в **`docs/agents/`**, рядом с **`docs/platform/`** и **`.cursor/rules/`**.

Если локально используется монорепозиторий **Ideas** (папка выше `solana-alpha`), не держите расходящиеся правки только в Ideas: при изменении контракта платформы или правил Cursor интегратор переносит изменения в этот репозиторий и делает push в **`v2`**.
